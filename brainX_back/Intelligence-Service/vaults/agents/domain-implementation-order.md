# Domain Implementation Order

## Domain Flow

구현 흐름은 `AI 설정 -> 검색/요약 -> 작성 보조 -> 대화형 탐색 -> 연결 추천 -> 정리 제안 -> 구조 분석 -> 인사이트 리포트` 순서로 본다.

먼저 사용자가 즉시 체감하는 노트 단위 기능을 만들고, 이후 전체 지식 베이스를 분석하거나 장기 작업을 생성하는 기능으로 확장한다.
