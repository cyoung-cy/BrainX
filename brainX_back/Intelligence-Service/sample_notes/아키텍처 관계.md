| 위치                                | 방향  | 넣을 말                              |
| --------------------------------- | --- | --------------------------------- |
| React -> API Gateway              | 오른쪽 | `HTTP 요청 / REST API 호출`           |
| API Gateway -> React              | 왼쪽  | `HTTP 응답 / JSON 응답`               |
| API Gateway -> MSA                | 오른쪽 | `요청 라우팅 / Reverse Proxy`          |
| MSA -> API Gateway                | 왼쪽  | `서비스 응답 반환`                       |
| API Gateway -> Eureka Server      | 아래  | `서비스 위치 조회 / Service Discovery`   |
| Backend Services -> Eureka Server | 왼쪽  | `서비스 등록 / Heartbeat`              |
| Backend Services -> Kafka         | 왼쪽  | `이벤트 발행 / Publish`                |
| Kafka -> Backend Services         | 오른쪽 | `이벤트 구독·소비 / Subscribe & Consume` |

그림에 바로 붙일 짧은 버전은 이게 제일 깔끔합니다.

```
React -> Gateway:
HTTP Request

Gateway -> React:
HTTP Response

Gateway -> MSA:
Route Request

MSA -> Gateway:
Service Response

Gateway -> Eureka:
Service Discovery

MSA -> Eureka:
Register / Heartbeat

MSA -> Kafka:
Event Publish

Kafka -> MSA:
Event Consume
```
