**방법 1 — 키워드 매칭** (Vector DB 불필요)

노트 제목이나 태그를 기준으로 같은 단어가 있으면 링크해요. 옵시디언의 기본 기능인 `[[링크]]`를 자동으로 찾아주는 플러그인(예: `Note Auto Creator`)이 이 방식이에요. 가볍고 빠르지만, "고양이"와 "펫"처럼 단어가 달라도 관련된 노트는 못 찾아요.

**방법 2 — Vector DB** (핵심 방법)

노트 전체를 벡터로 변환해서 저장하고, 의미가 비슷한 노트끼리 자동으로 연결해요. 노트가 많을수록 빛을 발하고, "머신러닝"과 "딥러닝" 같이 단어는 달라도 주제가 비슷하면 찾아줘요.

**방법 3 — LLM 직접 호출** (Vector DB 없이도 가능)

노트 내용을 AI(Claude, GPT 등)에게 보내서 "이 노트와 연결할 만한 노트 찾아줘"라고 물어봐요. 가장 똑똑하지만, 노트 수백 개를 전부 AI에 보내면 비용과 시간이 많이 들어요.
