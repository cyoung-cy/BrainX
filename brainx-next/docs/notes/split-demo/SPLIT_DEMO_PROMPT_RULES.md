# SPLIT_DEMO_PROMPT_RULES.md

## 목적

이 문서는 Claude Code에게 `/editor-lab/split-demo` 관련 작업을 요청할 때 사용하는 프롬프트 작성 규칙이다.

모든 split-demo 작업 요청은 이 문서의 규칙을 따른다.

---

# 기본 컨텍스트

대상 페이지:

http://localhost:3000/editor-lab/split-demo

설명:

- BrainX 노트 작성 페이지 프로토타입
- 실제 `/notes/:id` 로 발전 예정
- 단순 UI 테스트 페이지 아님
